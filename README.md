# Trabalho: Mecanica Do Voo
Repositorio com a implementação de um simulador 6DOF para análise de estabilidade dinâmica da aeronave EMB110 - Bandeirante.

## Modelo físico/matemático
A modelagem matemática na formulação de variáveis de estado é descrita em Aircraft Flight Dynamics and Control - Wayne Durham, e considera uma aeronave com variação de massa desprezível, indeformável (hipótese de corpo rígido) e aproximação de Terra plana (desprezados os efeitos da curvatura da Terra, como efeito Coriolis e aceleração centrífuga).
## Modelo Aerodinâmico
A modelagem aerodinâmica da aeronave foi feita no software Athena Vortex Lattice, considerando aerodinâmica linear de regime estacionário. Também foram incluídos os efeitos causados pelos perfis aerodinâmicos da aeronave, por meio de análise aerodinâmica no software XFLR5 (método dos painéis).
