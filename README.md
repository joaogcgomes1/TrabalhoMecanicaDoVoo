# Trabalho: Mecanica Do Voo
Repositorio com a implementação de um simulador 6DOF para análise de estabilidade dinâmica da aeronave EMB110 - Bandeirante.

A modelagem matemática na formulação de variáveis de estado é descrita em Aircraft Flight Dynamics and Control - Wayne Durham. A modelagem aerodinâmica da aeronave foi feita no software Athena Vortex Lattice, considerando aerodinâmica linear de regime estacionário. Também foram incluídos os efeitos causados pelos perfis aerodinâmicos da aeronave, por meio de análise aerodinâmica no software XFLR5 (método dos painéis).

# Resultados e discussões
A metodologia e as discussões do comportamento dinâmico da aeronave estão descritos em detalhes no PDF MecVoo202502_JoaoGabrielCerqueiraGomes.pdf
